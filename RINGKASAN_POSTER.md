## Pemahaman Proyek NutriMate

**NutriMate** adalah aplikasi pendamping nutrisi pribadi berbasis Android yang dirancang untuk membantu pengguna mengelola pola makan berdasarkan kondisi medis mereka.

### Fitur Utama yang Dipahami:

1. **Manajemen Profil Pengguna**
   - Menghitung target kalori harian (TDEE)
   - Berdasarkan usia, berat badan, tinggi badan, jenis kelamin, dan tingkat aktivitas

2. **Pelacakan Kondisi Medis**
   - Diabetes: Peringatan asupan gula
   - Hipertensi: Pemantauan sodium
   - Kolesterol: Tracking lemak
   - Gastritis: Monitoring makanan pedas

3. **Pencatatan Makanan**
   - Database makanan Indonesia lokal (Nasi Goreng, Tempe, Tahu, Sate, dll)
   - 15+ makanan Indonesia yang sudah terdaftar
   - Tracking untuk Sarapan, Makan Siang, Makan Malam, dan Camilan

4. **Dashboard Pintar**
   - Pelacakan kalori real-time
   - Breakdown makronutrien (karbohidrat, protein, lemak)
   - Peringatan kesehatan yang dipersonalisasi
   - Target asupan air

5. **Arsitektur Offline-First**
   - Room Database (SQLite)
   - Semua data tersimpan lokal
   - Privasi data terjamin

### Teknologi yang Digunakan:
- **Bahasa**: Kotlin
- **Framework**: Android Native
- **UI**: XML Layouts & Jetpack Compose
- **Arsitektur**: MVVM
- **Database**: Room Database
- **Format Data**: JSON

## Cara Melihat Poster

### Di GitHub:
1. Buka repository: https://github.com/TrendoD/Nutri-Mate
2. Scroll ke bagian "Promotional Poster" di README
3. Klik pada file poster untuk melihat versi full-size

### Lokal:
1. Clone repository
2. Buka file `POSTER_NUTRIMATE.svg` di browser
3. Atau buka file PNG dengan image viewer

### Edit/Kustomisasi:
1. Download `POSTER_NUTRIMATE.svg`
2. Buka dengan Inkscape (gratis) atau Adobe Illustrator
3. Edit sesuai kebutuhan
4. Export ke format yang diinginkan
